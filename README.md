# TechInfra - Página Web Informática

Página web moderna dedicada a la informática, infraestructura, virtualización, automatización y ciberseguridad.

## Archivos

- `index.html`: estructura principal de la página.
- `assets/styles.css`: estilos y animaciones.
- `assets/script.js`: animación tipo Matrix y efectos al hacer scroll.

## Cómo desplegar en GitHub Pages

1. Crea un repositorio en GitHub.
2. Sube todos los archivos de este proyecto.
3. Ve a **Settings** > **Pages**.
4. En **Build and deployment**, selecciona:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Guarda los cambios.
6. GitHub generará una URL pública para tu sitio.

## Personalización rápida

En `index.html`, cambia:

- Nombre del sitio: `TechInfra`
- Correo de contacto: `contacto@tudominio.com`
- Servicios o tecnologías según tu necesidad.
