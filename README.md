# Portafolio Emanuel Chacón

Portafolio personal con diseño oscuro/neón, animaciones, partículas y sección de proyectos destacados.

## Publicar en GitHub Pages

1. Descomprime este ZIP.
2. Sube `index.html`, `style.css`, `script.js` y `README.md` a tu repositorio.
3. En GitHub entra a **Settings > Pages**.
4. Selecciona **Deploy from a branch**.
5. Branch: `main` y carpeta `/root`.
6. Guarda y espera la URL `https://usuario.github.io/repositorio/`.
