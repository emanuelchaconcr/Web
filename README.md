# Portafolio Personal de Informática

Sitio web tipo portafolio personal con animaciones, estilo tecnológico y secciones para:

- Presentación personal
- Sobre mí
- Habilidades
- Proyectos
- Experiencia
- Contacto

## Publicar en GitHub Pages

1. Crea un repositorio en GitHub.
2. Sube estos archivos descomprimidos.
3. Ve a Settings > Pages.
4. Selecciona Deploy from a branch.
5. Elige main y /root.
6. Guarda los cambios.

La web quedará disponible en:

https://TUUSUARIO.github.io/NOMBRE-DEL-REPOSITORIO/
