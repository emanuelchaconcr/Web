# Portafolio Emanuel Chacón

Portafolio personal de infraestructura TI listo para publicar en GitHub Pages.

## Publicar en GitHub Pages

1. Descomprime este ZIP.
2. Sube `index.html`, `style.css`, `script.js` y `README.md` a tu repositorio.
3. En GitHub ve a **Settings > Pages**.
4. Selecciona **Deploy from a branch**.
5. Branch: `main` y carpeta `/root`.
6. Guarda y espera unos minutos.

Tu web quedará en:

`https://TUUSUARIO.github.io/NOMBRE-DEL-REPOSITORIO/`
